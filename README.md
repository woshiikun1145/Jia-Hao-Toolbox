# 嘉豪工具箱 (Jiahao Toolbox) 3.0

一个专为教学环境设计的便捷工具集合，提供快速启动教学软件的功能。

核心工具
一键启动白板和展台 - 快速启动教学所需软件
简洁GUI界面 - 无需复杂操作，点击即可使用

 基础使用
1. 下载并运行 `嘉豪工具箱.exe`
2. 在主界面点击相应按钮启动需要的功能：
   - 启动白板和展台
   - 仅启动白板  
   - 仅启动展台
   - 启动希沃白板

控制台命令
在"控制台"区域输入以下命令获得更多功能：

基础命令：
- ver - 显示版本信息
- help - 查看帮助文档
- space - 打开作者B站主页
- command - 显示完整命令列表
- debug - 打开调试模式

系统要求
- 操作系统: Windows 7/10/11
- Python版本: 3.6+ (仅源码版本需要)
- 依赖: 标准tkinter库

常见错误：
0x00 - 启动失败
0x01 - 无效的命令
1x00 - 文件缺失
1x00 - 执行命令时出现未知的意外错误
2xf0 - S模式已开启
2xf1 - 未在调试模式下执行需在调试模式下执行的命令
